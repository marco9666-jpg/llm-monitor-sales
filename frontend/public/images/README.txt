# 圖片放置說明

為了讓網站正確顯示你的 TokenMeter 軟體截圖，請按照以下步驟操作：

## 📁 圖片存放位置

所有圖片請放在這個資料夾：
```
Desktop/llm-monitor-sales/frontend/public/
```

## 🖼️ 需要準備的圖片（共 3 張）

### 1. App Icon（軟體圖示）
- **檔名**：`app-icon.png`
- **來源**：你傳給我的第一張圖（藍色方形圖示）
- **用途**：網站 Logo、瀏覽器分頁圖示

### 2. 設定頁面截圖
- **檔名**：`screenshots/settings.png`
- **來源**：你傳給我的第二張圖（設定頁面深色截圖）
- **用途**：展示軟體設定功能

### 3. 多窗口總覽截圖
- **檔名**：`screenshots/overview.png`
- **來源**：你傳給我的第三張圖（多窗口組合截圖）
- **用途**：主展示區大圖

### 4. 懸浮窗截圖
- **檔名**：`screenshots/floating.png`
- **來源**：你傳給我的第四張圖（懸浮數據窗截圖）
- **用途**：展示即時監控功能

## ✅ 操作步驟

1. 把 4 張圖片從訊息/下載資料夾中複製出來
2. 第一張圖（App Icon）改名為 `app-icon.png`，放到 `frontend/public/` 底下
3. 其餘 3 張圖分別改名，放到 `frontend/public/screenshots/` 底下
4. 回到瀏覽器，重新整理 http://localhost:5173

## 💡 小提示

- macOS 的視窗截圖（Cmd + Shift + 4 + 空白鍵）會自動帶有漂亮的陰影和圓角，直接放上去就很好看
- 圖片格式建議使用 PNG（保留透明背景）
- 如果圖片太大，可以先用預覽程式壓縮一下尺寸

放完圖片後，網站就會自動顯示你的軟體截圖了！
